### Author: Syed Maqbool

## **Task 4 - Stock Market Prediction using Numerical and Textual Analysis**
In this exercise, we will Create a hybrid model for stock price/performance 
prediction using numerical analysis of historical stock prices, and 
sentimental analysis of news headlines. 


The steps are:
1.   Import libraries
2.   Load Dataset
3.   Data Exploration
4.   Data Visualization
5.   Data Preparation
6.   Train Data
7.   Test Data
8.   Predicting The Score 
9.   Model Evaluation
